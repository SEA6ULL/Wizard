[COLORdeepskyblue][B]Welcome to the SEA6ULL Wizard.[/B][/COLOR]

All builds require Kodi 21.0 or greater.

No content is hosted or provided. Premium debrid account required.
MDBList API key required. Find yours at mdblist.com/preferences API Access

Builds will not function properly until credentials have been authorized.
Widgets will not display properly until MDBList API key has been entered.

Navigate to the Settings menu for POV and authorize debrid provider and 
MDBList API key.

Youtube (for trailers) requires device specific installation, therefore it is not
included within builds. The first time you try to play a trailer you will be 
prompted to install Youtube. It will crash Kodi after it installs. Force close Kodi
and restart. The next time you try a trailer the setup wizard will help you 
configure Youtube settings, then you're good to go.
